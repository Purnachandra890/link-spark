# 🔗 Link Spark – URL Shortener

Link Spark is a simple and fast URL shortener web application that allows users to convert long URLs into short, shareable links.

---

## 🚀 Features

* 🔗 Shorten long URLs instantly
* 📋 Copy short links easily
* ⚡ Fast and responsive UI
* 🌐 Backend API for URL handling
* 📊 (Optional) Track clicks (if implemented)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
